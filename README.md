# Base45Lite

An implementation of [The Base45 Data Encoding](https://datatracker.ietf.org/doc/draft-faltstrom-base45/)

## Installation

```sh
gem install base45_lite
```

## Usage

See [test/test_base45_lite.rb](test/test_base45_lite.rb)
